--- title: makeUrl method: model.makeUrl( type, [parent] ) returnType: String returnDescription: The generated URL complete: false category: model ---

Generates a URL based on this model's url, the parent model of this model, the type of request we're making and Falcon's defined baseApiUrl

type
The type of request we're making (GET, POST, PUT, DELETE)
parent
Optional override of the model's parent to generate the url with. If parent is 'null' then this model will act as the root node.
Falcon.baseApiUrl = "http://www.falconjs.com/"; var Dog = Falcon.Model.extend({'url': "dog.json"}); var Owner = Falcon.Model.extend({'url': "owner.json"}); var owner = new Owner({"id": 1}); var dog = new Dog({"id": 2}, owner); // owner is the 'parent' model console.log( dog.makeUrl("GET") ); // => http://www.falconjs.com/owner/1/dog/2.json console.log( dog.makeUrl("POST") ); // => http://www.falconjs.com/owner/1/dog.json console.log( dog.makeUrl("PUT") ); // => http://www.falconjs.com/owner/1/dog/2.json console.log( dog.makeUrl("DELETE") ); // => http://www.falconjs.com/owner/1/dog/2.json console.log( dog.makeUrl("GET", null) ); // => http://www.falconjs.com/dog/2.json
