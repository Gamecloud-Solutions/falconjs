--- title: isNew method: model.isNew() returnType: Boolean returnDescription: Is this a new model? complete: false category: model ---

Method used to determine if this model is 'new' or not. A model is deemed new if no id has been set (it's still null).

var Dog = Falcon.Model.extend({ defaults: { "name": "Fido" } }); var dog = new Dog() console.log( dog.isNew() ); // => true dog.set("id", 1) console.log( dog.isNew() ); // => false
