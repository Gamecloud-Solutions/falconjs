--- title: get method: model.get( attribute ) returnType: mixed returnDescription: The value at the specified attribute complete: false category: model ---

Method used to look up the value of an attribute on this model. If the attribute is an observable, then the observable is unwrapped and its internal value is returned.

attribute
The attribute to look up
var Dog = Falcon.Model.extend({ defaults: { "name": "Fido" }, observables: { "hello": "world" } }); var dog = new Dog(); console.log( ko.isObservable( dog.name ) ); // => false console.log( ko.isObservable( dog.hello ) ); // => true console.log( dog.get("name") ); // => Fido console.log( dog.get("hello") ); // => world
