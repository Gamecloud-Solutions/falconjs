--- title: isCollection method: Falcon.isCollection( object ) returnType: Boolean returnDescription: Is the object a collection complete: false category: Utility ---

Method used to test if an object is a Falcon Collection.

object
The Object to test
var array = []; var object = new Falcon.Object(); var collection = new Falcon.Collection(); var model = new Falcon.Model(); var view = new Falcon.View(); console.log( Falcon.isCollection(array) );// => false console.log( Falcon.isCollection(object) );// => false console.log( Falcon.isCollection(model) );// => false console.log( Falcon.isCollection(collection) ); // => true console.log( Falcon.isCollection(view) );// => false
