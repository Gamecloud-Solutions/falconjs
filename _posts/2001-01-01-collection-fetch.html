--- title: fetch method: collection.fetch( [options] ) returnType: XmlHttpRequest returnDescription: The XmlHttpRequest created complete: false category: collection ---

Shorthand method for a GET sync request to get all of the models from the server.

options
The options to pass to the 'sync' method
var Dog = Falcon.Model.extend({url: 'dog.json'}); var Dogs = Falcon.Collection.extend({model: Dog}); var dogs = new Dogs() dogs.fetch() // 'GET' request to '/dog.json'
