--- title: apply method: Falcon.apply( root, [element], [callback] ) returnType: undefined returnDescription: undefined complete: false category: Utility ---
root
A view or an observable containing a view to be applied to the application element
element
The element to intialize the application in. If none is provided, the body element is used.
callback
Method called after the bindings have been initialized and applied.

The method to initialize a Falcon application and apply its bindings against an initial view.
