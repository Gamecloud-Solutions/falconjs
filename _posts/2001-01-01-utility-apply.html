--- title: apply method: Falcon.apply( root, [element], [callback] ) returnType: undefined returnDescription: undefined complete: false category: Utility ---

nitializes a Falcon application and applies its bindings against an initial view.

root
A view or an observable containing a view to be applied to the application element
element
The element to initialize the application in. If none is provided, the body element is used.
callback
Method called after the bindings have been initialized and applied.
var LayoutView = Falcon.View.extend({ "url": "#layout_view_tmpl" }); Falcon.apply( new LayoutView ); // Injects the layout view into the 'body' tag Falcon.apply( new LayoutView, "#application" ); // Injects the layout view into the element with id="application" // apply can also handle observables var viewObs = ko.observable(); Falcon.apply( viewObs ); // Applies the observable to the 'body' tag, nothing is currently rendered viewObs( new LayoutView ); // Renders a layout view in 'body'
