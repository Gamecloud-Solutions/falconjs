--- title: isDataObject method: Falcon.isDataObject( object ) returnType: Boolean returnDescription: Is the object a DataObject (Model or Collection)? complete: false category: Utility ---

Method used to test if an object is a Falcon Model or Collection

object
The Object to test
var array = []; var object = new Falcon.Object(); var collection = new Falcon.Collection(); var model = new Falcon.Model(); var view = new Falcon.View(); console.log( Falcon.isDataObject(array) );// => false console.log( Falcon.isDataObject(object) );// => false console.log( Falcon.isDataObject(model) );// => true console.log( Falcon.isDataObject(collection) ); // => true console.log( Falcon.isDataObject(view) );// => false
