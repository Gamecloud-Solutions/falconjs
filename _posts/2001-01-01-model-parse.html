--- title: parse method: model.parse( data, [options], [xhr] ) returnType: Object returnDescription: Parsing on a model expects an object to be returned complete: false category: model ---

Parses the response data from an XmlHttpRequest request. A json object must be returned from this method in order for the data to be filled back into the model properly. This is useful if you're in need of converting a server response into a different format before attempting to fill all of the model's data.

data
The raw response data
options
The options fed initially into the 'sync' request
xhr
The XmlHttpRequest object
// ------------------------------------- // Example Server Response JSON for /dog.json // ------------------------------------- { data: { "id": 1, "name": "Fido" }, created_at: "01/01/1970", updated_at: "01/01/1970" } // ------------------------------------- // Javascript // ------------------------------------- var Dog = Falcon.Model.extend({ 'url': "dog.json", defaults: { "name": "" } // Example parse method, the object that is returned from // this will be passed into the model's fill() method. parse: function( response ) { return response["data"]; } }); var dog = new Dog(); // Get's the server data and runs the parse method // Callback is called on complete dog.fetch(function() { console.log( dog.get("id") ); // => 1 console.log( dog.get("name") ); // => Fido });
