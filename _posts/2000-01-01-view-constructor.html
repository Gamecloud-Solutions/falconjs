--- title: initialize method: new Falcon.View() complete: false category: View ---

nitializes this class, this is called within the default constructor and is expected to be overridden by inheriting classes. Internally this also called the Falcon.Object constructor to in initialize any 'defaults' or 'observables'.

var DogView = Falcon.View.extend({ initialize: function() { console.log("nitialized the Dog View"); } }); new DogView(); // => nitialized the Dog View
