--- title: isView method: Falcon.isView( object ) returnType: Boolean returnDescription: Is this a Falcon view? complete: false category: Utility ---

Method used to test if an object is a Falcon View.

object
The Object to test
var array = []; var object = new Falcon.Object(); var collection = new Falcon.Collection(); var model = new Falcon.Model(); var view = new Falcon.View(); console.log( Falcon.isView(array) );// => false console.log( Falcon.isView(object) );// => false console.log( Falcon.isView(model) );// => false console.log( Falcon.isView(collection) ); // => false console.log( Falcon.isView(view) );// => true
