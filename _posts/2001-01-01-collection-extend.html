--- title: extend method: Falcon.Collection.extend( protoProps, [staticProps] ) returnType: undefined returnDescription: undefined complete: false category: collection ---

Method used to create a new collection that inherits from the base Falcon Collection.

protoProps
Properties for the prototype of a class
staticProps
Static properties for a given object
var Dog = Falcon.Model.extend(); var Dogs = Falcon.Collection.extend({ model: Dog, bark: function() { console.log("All the dogs bark"); } },{ say: function() { console.log("Woof!"); } }); var dogs = new Dogs(); Dogs.say(); // => Woof! dogs.bark(); // => "All the dogs bark";
