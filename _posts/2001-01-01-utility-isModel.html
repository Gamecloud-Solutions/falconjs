--- title: isModel method: Falcon.isModel( object ) returnType: Boolean returnDescription: Is the object a model? complete: false category: Utility ---

Method used to test if an object is a Falcon Model.

object
The Object to test
var array = []; var object = new Falcon.Object(); var collection = new Falcon.Collection(); var model = new Falcon.Model(); var view = new Falcon.View(); console.log( Falcon.isModel(array) );// => false console.log( Falcon.isModel(object) );// => false console.log( Falcon.isModel(model) );// => true console.log( Falcon.isModel(collection) ); // => false console.log( Falcon.isModel(view) );// => false
