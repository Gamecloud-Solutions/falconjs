--- title: "off" method: object.off( event, [callback] ) returnType: Falcon.Object returnDescription: This instance complete: false category: Object ---

Removes event listening for a specific event. Optionally, if a callback is given, only that callback will be removed from the list of calbacks for the specified event.

event
The event to stop listening for.
callback
The speicifc callback function to remove.
var obj = new Falcon.Object(); obj.on("alert", function() { alert("Hello World"); }); obj.trigger("alert"); obj.off("alert"); obj.trigger("alert"); var obj = new Falcon.Object(); var alert_callback_one = function() { alert("Alert One"); }; var alert_callback_two = function() { alert("Alert Two"); }; obj.on("alert", alert_callback_one); obj.on("alert", alert_callback_two); obj.trigger("alert"); obj.off("alert", alert_callback_one); obj.trigger("alert");
