--- title: set method: model.set( attribute, [value] ) returnType: Falcon.Model returnDescription: This instance complete: false category: model ---

Sets a value for the specific attribute, creating one if it does nto exist. If an object is passed into the first argument of this method, each of the object's key value pairs will be set on this model.

attribute
The attribute to look up
value
The value to assign
values
An object of values to set
var Dog = Falcon.Model.extend({ url: "dog.json", defaults: { "name": "", "color": "" } }); var dog = new Dog({'id': 1}); dog.set("name", "Fido"); dog.set("color", "Black"); console.log( dog.get("name") ); // => Fido console.log( dog.get("color") ); // => Black dog.set({ "name": "Spot", "color": "Tan" }); console.log( dog.get("name") ); // => Spot console.log( dog.get("color") ); // => Tan
