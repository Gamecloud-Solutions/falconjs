Initializes this class, this is called within the default constructor and is expected to be overridden by inheriting classes. Internally this also called the Falcon.Object constructor to in initialize any 'defaults' or 'observables'. Any arguments that are passed into the native constructor will also be passed in as the arguments for the initialize method.

{% highlight javascript linenos %} var DogView = Falcon.View.extend({ initialize: function() { console.log("Initialized the Dog View"); } }); new DogView(); // => Initialized the Dog View {% endhighlight %} {% assign methodName = "extend" %} {% assign methodExample = "Falcon.View.extend( protoProps, [staticProps] )" %} {% include methodOverview.html %}

protoProps

The url or element id where this template is defined. Can either be a string or a function and can either point to a uri or a DOM object identifier. This attribute should be overridden by any inheriting classes

{% highlight javascript linenos %} var DogUrlView = Falcon.View.extend({ url: "dog.tmpl" }); var DogIdView = Falcon.View.extend({ url: "#dog" }); var DogFunctionView = Falcon.View.extend({ url: function() { return "dog.html" } }); var dog_url_view = new DogUrlView(); var dog_id_view = new DogIdView(); var dog_function_view = new DogFunctionView(); Falcon.baseTemplateUrl = "http://www.falconjs.com/"; console.log( dog_url_view.makeUrl() ); // => http://www.falconjs.com/dog.tmpl console.log( dog_id_view.makeUrl() ); // => #dog console.log( dog_function_view.makeUrl() ); // => http://www.falconjs.com/dog.html {% endhighlight %} {% assign methodName = "display" %} {% assign methodExample = "view.display()" %} {% include methodOverview.html %}

Executed when this view is displayed (added to the DOM). This method is called from the view binding when rendering this view's template is complete and added to the DOM. This should be overridden by inheriting classes and should primarily be used for 'on-display' functionality such as certain event handlers

{% highlight javascript linenos %} var DogView = Falcon.View.extend({ url: '#dog_view', display: function() { console.log("Dog View Displayed") } }); var view_model = { showing_dog_view: ko.observable(false), my_dog_view: new DogView() }; console.log("Initializing"); setTimeout(function() { view_model.show_dog_view(true) }, 5000); Falcon.apply(view_model, '#application'); {% endhighlight %}

Executed when this view is disposed (removed from the DOM). This method is called from the unrender method which itself is called from the 'view' binding. This should be overridden by inheriting classes and should primarily be used for removing any 'on-display' functionality that was added in the display method

{% highlight javascript linenos %} var DogView = Falcon.View.extend({ url: '#dog_view', dispose: function() { console.log("Dog View Displayed No Longer") } }); var view_model = { showing_dog_view: ko.observable(true), my_dog_view: new DogView() }; console.log("Initializing"); setTimeout(function() { view_model.show_dog_view(false) }, 5000); Falcon.apply(view_model, '#application'); {% endhighlight %}

An observable that specifies if the view has loaded not. A view is loaded when its template is found and stored into the template cache. Making this an observable allows us to bind against it and for the 'view' binding to update properly.

{% assign methodName = "makeUrl" %} {% assign methodExample = "view.makeUrl()" %} {% assign returnType = "String" %} {% assign returnDescription = "The full url to the template" %} {% include methodOverview.html %}

Method used to 'intelligently' make a url to point to the remote version of this template. If the URL begins with a '#' it will leave the url as is because we're identifying an on-page template element to look up rather than a remote resource.

{% highlight javascript linenos %} var DogUrlView = Falcon.View.extend({ url: "dog.tmpl" }); var DogIdView = Falcon.View.extend({ url: "#dog" }); var dog_url_view = new DogUrlView(); var dog_id_view = new DogIdView(); console.log( dog_url_view.makeUrl() ); // => /dog.tmpl console.log( dog_id_view.makeUrl() ); // => #dog Falcon.baseTemplateUrl = "http://www.falconjs.com/"; console.log( dog_url_view.makeUrl() ); // => http://www.falconjs.com/dog.tmpl console.log( dog_id_view.makeUrl() ); // => #dog {% endhighlight %}
