Sets a specific property of all of the models in the collection to a certain value.

key
The property to update.
value
The value to set on each model at the given key.
{% highlight javascript linenos %} var Dog = Falcon.Model.extend(); var Dogs = Falcon.Collection.extend({model: Dog}); var dogs = new Dogs([ {"name": "Fido"}, {"name": "Skip"} ]); console.log( dogs.pluck("name") ); // [Fido, Skip] dogs.set("name", "Super Dog"); console.log( dogs.pluck("name") ); // [Super Dog, Super Dog] {% endhighlight %}
