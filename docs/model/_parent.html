This represents the 'parent' object that holds this model. Generally this is used for determining the URL structure of rest objects in the makeURL() routine. This should also be a Model.

{% highlight javascript linenos%} var Dog = Falcon.Model.extend({"url": "dog.json"}); var Owner = Falcon.Model.extend({"url": "owner.json"}); var owner = new Owner(); var dog = new Dog( owner ); // Owner becomes the 'parent' of dog console.log( dog.parent === owner ); // true {% endhighlight %}
